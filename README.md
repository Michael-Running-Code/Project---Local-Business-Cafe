# Good Café

A cozy local café website project.


## Features
- Responsive design (mobile + desktop)
- Menu filtering (All / Bake / Drink)
- Staff carousel on About page
- Google Maps embed on Contact
- interactive menu on mobile

## Technologies
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
